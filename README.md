# Laundry Kilat - Landing Page

Landing page jasa laundry antar jemput dengan fitur kalkulator harga otomatis berbasis JavaScript.

## Live Demo
[Lihat Demo](https://laundry-kilat.netlify.app)

## Fitur
- Kalkulator harga otomatis
- Responsive design
- Order via WhatsApp
- Animasi scroll

## Teknologi
- HTML5
- Tailwind CSS (CDN)
- JavaScript (Vanilla)
